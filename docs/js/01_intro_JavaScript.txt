O JavaScript e uma linguagem de programação que foi feita para adicionar interactividade nas páginas web. 
Isto pode ser reconhecido em jogos que correm no browser a 2D e a 3D e em animações do site, como também no uso de base de dados. 

Os programadores contruiram uma lista de ferramentas para facilitar o uso do JavaScript, tais como: 

- As APIs: Ajudam ao desenvolvimento de aplicações para criar sites dinamicos, manipular video a partir de camaras-web,
para gerar graficos 3D, bases de dados e muito mais. 

- APIs externas: que ajudam aos programadores incorporar funcionalidades externas tais como do twitter ou do facebook.

- Frameworks e livrarias para acelarar o desenvelvimento de sites e aplicações.
