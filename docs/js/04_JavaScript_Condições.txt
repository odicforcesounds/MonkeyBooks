CONDIÇÕES

As "condições" servem para estruturar código de modo a testar se uma ou mais expressões ( dados de variaveis ) são
verdadeiros ou falsos. 

let gelado = 'chocolate'
if(gelado === 'chocolate') {
  console.log('adoro gelado de chocolate!'); 
} else {
  console.log('que pena, não têm gelado de chocolate...'); 
}
