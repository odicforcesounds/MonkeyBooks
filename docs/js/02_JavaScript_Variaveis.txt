VARIAVEIS 

Se os dados fossem estáticos, nada poderia ser alterado e como tal, tudo seria estático ( tal como os sites feitos em
HTML ) sem CSS e sem JavaScript. 

As variaveis em JavaScript podem ser definidas e\ou declaradas das seguintes maneiras: 

1. -> var variavel01 = 1
2. -> let variavel01 = 1
3. -> const variavel01 = 1

Nota: No exemplo que se usa "var" para definir a variavel 'variavel01', o JavaScript permite que a declaração seja feita
continuamente ( repetidamente ) com o mesmo conteúdo ou outro qualquer.. No exemplo que se usa "let" para definir a
variavel 'variavel01', o JS não permite que a declaração da variavel já pré-declarada, seja feita duas ou mais vezes,
sendo possível atribuir outro valor à variavel. No exemplo que se usa o "const" para definir a variavel 'variavel01', o
JS não permite a repetição da mesma declaração como também o valor da variavel não pode mudar. ( const -> Constante ) 

As variaveis podem conter vários tipos de dados, sendo estes os seguintes: 

1. -> Strings -> let variavel = 'Oscar'
2. -> Numeros -> let variavel = 123
3. -> Boolean -> let variavel = true / false 
4. -> Array -> let variavel = ['macaco', 32, 'musica'] 
5. -> Objecto -> let variavel = document.querySelector('h1')
